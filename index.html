    • What is UI?
  • The UI in UI design stands for user interface
  • The user interface is the graphical layout of an application.
  • It consists of the buttons users click on, the text they read, the images, sliders, text entry fields, and all the rest of the items the user interacts with.
  • This includes screen layout, transitions, interface animations and every single micro-interaction.
  • Any sort of visual element, interaction, or animation must all be designed.
    • What is UX?
  • UX stands for user experience
  • A user's experience of the app is determined by how they interact with it.
  • Is the experience smooth and intuitive or clunky and confusing?
  • Does navigating the app feel logical or does it feel arbitrary?
  • Does interacting with the app give people the sense that they're efficiently accomplishing the tasks they set out to achieve or does it feel like a struggle?
    • UI v/s UX

    A UX designer decides how the user interface works while the UI designer decides how the user interface looks.


    User experience is determined by how easy or difficult it is to interact with the user interface elements that the UI designers have created.

    • Typography
  • Everytime you see a font, your brain will activate the visual traits
    • Fonts Classifications

  • Choose a font that fits your context
  • Long thin lines convey Beauty
  • Bold fonts are powerful and masculine
  • Simple fonts convey Directness
  • Complex fonts convey Uniqueness
  • Slanted Fonts convey fast movement
    • Font Classifications

  • Sans-Serifs Are More Readable Via Screens
  • Serifs Are More Elegant and Scientific
  • Lowercase Conveys Compassion and Innovation
  • Uppercase Conveys Power and Strength
  • Condensed Fonts Convey Tightness and Precision
  • Spacious Fonts Convey Relaxation
    • Typefaces

  • Typeface is a family of fonts
  • Instead of using many different fonts, you should use few similar fonts
  • Using Whitespace is as important as text.
  • You should create a visual hierarchy using proper headings and sub-headings
  • Example: Medium
    • Iconography

  • Icons are widely used in websites/apps to convey meaning
  • While using icons make sure they are uniform and clear
  • If it takes more than 5 seconds to think of an icon, it will probably be ineffective.
  • Always have at least 1 cm x 1 cm space minimum around the icon for legibility and easy tapping on mobiles
    • Colour Psychology

  • Every website has a Primary Colour and a Secondary colour. While using colours keep in mind the 60:30:10 rule.
  • Don't use absolute black colour, because it strains the eyes, rather use darker shades of colours
  • Websites should contain a light and dark mode so that users can choose what they need.
  • Colours determine brands and their Psychology, therefore when building a brand you should focus on the colour as well.
    • Consistency

  • Your designs should be consistent, i.e. if you are using a colour scheme, you should be using it throughout your website.
  • If you are using Serif fonts, you can't switch to sans-serifs later
  • You will also have to be consistent with brand guidelines.
    • Responsiveness

  • Since, there are devices of varied sizes, you have to keep them in mind while creating a website.
  • Users behave differently with different websites.
  • You have to create it for a mobile, a tablet as well as a laptop screen.
  • Therefore, the content has to be properly ordered.
    • Accessibility

  • What about those users who have one arm or are visually impaired?
  • What about the users who come from areas with low literacy rates?
  • What about users with poor connectivity or low phone storage?
  • There should be enough colour contrast that users with visual impairments can also distinguish
  • Can you navigate your website without using a mouse?
    • Basics of UX
  • Home Page is the first point of entry for a user
  • There are various pages you can navigate to from the Home Page like for example, the login page, contents page etc.
  • It should be easy for the user to navigate the website and reach where he wants to.

    There are various laws that can help you design the layout of the website and help the user navigate the website with ease.
